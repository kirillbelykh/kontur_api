# Kontur API

Windows-приложение для рабочих сценариев Контур.Маркировки: заказ кодов,
загрузка кодов, ввод в оборот, задания на ТСД, агрегация и печать этикеток.

В проекте есть два ярлыка запуска:

- `KonturTestAPI` - текущий интерфейс UI v2 на PyWebView.
- `KonturAPI` - старый интерфейс на CustomTkinter.

## Установка в один клик

1. Склонируйте проект с GitHub.
2. Откройте папку проекта.
3. Дважды кликните `setup.bat`.

Установщик подготовит компьютер:

- установит Git через `winget`, если Git отсутствует;
- установит `uv` через `winget` или официальный fallback-установщик;
- установит Python 3.12 и зависимости проекта в `.venv`;
- создаст `.env` из `.env.example`, если `.env` ещё нет;
- создаст рабочие папки на рабочем столе: `Коды км`, `Агрег коды км`, `Удаленные`;
- найдёт или установит Яндекс.Браузер;
- скачает подходящий `driver/yandexdriver.exe`;
- создаст ярлыки `KonturTestAPI` и `KonturAPI` на рабочем столе.

BarTender и CryptoPro/CAdES - внешние лицензируемые программы. Установщик
проверяет наличие BarTender SDK и предупреждает, если его нет, но не ставит
лицензионное ПО автоматически. Сертификат с закрытым ключом должен быть
установлен в хранилище текущего пользователя Windows.

## Локальная настройка

В репозитории хранится `.env.example`, а настоящий `.env` не коммитится. На
новом ПК установщик автоматически копирует `.env.example` в `.env`.

Если на новом ПК другой сертификат, оставьте `THUMBPRINT` пустым, чтобы
приложение выбрало первый подходящий сертификат, либо укажите нужный thumbprint
вручную.

## Ручной запуск

```powershell
uv python install 3.12
uv sync --python 3.12 --frozen
.\.venv\Scripts\python.exe ui_v2\main_v2.py
```

Старый интерфейс:

```powershell
.\.venv\Scripts\python.exe main.pyw
```

## Восстановление ярлыков

Если ярлыки на рабочем столе удалили или они сломались, запустите
`repair_shortcut.bat`.

## Обновление установленного проекта

```powershell
git pull origin main
uv sync --python 3.12 --frozen
```

После этого снова запускайте приложение через ярлык `KonturTestAPI`.

## Проверки для разработки

```powershell
uv sync --python 3.12 --group dev
python -m unittest tests.test_ui_v2_api_bridge
python -m py_compile ui_v2\api_bridge.py ui_v2\main_v2.py
```
